# rgit

A cheap git clone written in Rust

## Features

- [x] Initialize the .git repository
- [x] Read a blob object
- [ ] Create a blob object
- [ ] Read a tree object
- [ ] Write a tree object
- [ ] Create a commit
- [ ] Clone a repository

## Tech

- rust v1.74.1
- flate2 v1.0.28
	- with features
		- zlib-ng
